# E-Com67 Platform Overview

E-Com67 is a modern e-commerce platform built on AWS using serverless architecture. The platform provides a comprehensive solution for online retail businesses.

## Key Features

- **Product Management**: Full CRUD operations for product catalog
- **Shopping Cart**: Persistent cart functionality with user sessions
- **Order Processing**: Automated order workflow with payment integration
- **AI-Powered Chat**: Intelligent customer support using Amazon Bedrock
- **Search Functionality**: Advanced product search with OpenSearch
- **Knowledge Base**: Document-based knowledge management for customer support

## Architecture

The platform uses a microservices architecture with:
- AWS Lambda for compute
- DynamoDB for data storage
- OpenSearch for search capabilities
- S3 for file storage
- Amazon Bedrock for AI features

## Customer Support

Our AI-powered chat system can help customers with:
- Product recommendations
- Order status inquiries
- General platform questions
- Technical support

The system uses retrieval-augmented generation (RAG) to provide accurate and contextual responses based on our knowledge base.
